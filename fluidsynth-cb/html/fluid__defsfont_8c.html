◆ new_fluid_defsfloader()

fluid_sfloader_t* new_fluid_defsfloader ( fluid_settings_t *  settings)

Creates a default soundfont2 loader that can be used with fluid_synth_add_sfloader(). By default every synth instance has an initial default soundfont loader instance. Calling this function is usually only necessary to load a soundfont from memory, by providing custom callback functions via fluid_sfloader_set_callbacks().

Parameters
settingsA settings instance obtained by new_fluid_settings()
Returns
A default soundfont2 loader struct
